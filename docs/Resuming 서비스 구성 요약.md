# AI 모의면접 서비스 구성 요약

## 서비스 개요

Resuming은 기술 면접관과 HR 면접관으로 구성된 
2명의 AI 아바타가 사용자의 이력서를 기반으로 맞춤형 질문을 진행하고, 
답변 내용과 음성 특징을 분석하여 개선 피드백을 제공하는 AI 모의면접 서비스입니다.

## 서비스 시나리오

로그인 → 기본 음성 등록 → 이력서 업로드
→ 이력서 정보 추출 → 기술·HR 맞춤 질문 생성
→ 2인 AI 면접관 모의면접 진행
→ 답변·음성 분석 → 결과 제공

## 시스템 아키텍처 및 구성

```text
[사용자]
```

## MVP 정의 및 설계

MVP는 고도화된 실시간 면접 기능보다 사용자가 실제로 모의면접을 진행하고 분석 결과를 확인할 수 있는 핵심 흐름 구현을 목표로 합니다.

### 핵심 기능

* 면접 세션 생성
* 이력서 PDF 업로드 및 텍스트 추출
* 이력서 기반 맞춤 질문 생성
* 음성 또는 텍스트 답변 입력
* 질문별 답변 저장 및 분석
* 답변 구체성, 논리성, 직무 연관성 평가
* 말하기 속도, 떨림, 추임새 등 음성 분석
* 질문별 피드백과 전체 총평 제공

### MVP 제외 및 추후 확장

실시간 립싱크, 실시간 꼬리 질문, 사용자 간 면접 대결, 기업별 면접 데이터 연동, 이전 면접과의 정밀 비교 기능은 추후 확장

## 핵심 기술

* **Frontend:** React, Vite
* **Backend:** FastAPI
* **Database:** PostgreSQL
* **Document:** PyPDF2
* **AI:** LLM 기반 질문 생성 및 답변 평가
* **Avatar:** 기술 면접관·HR 면접관 아바타 영상 및 상황별 전환
* **Voice:** STT, WPM, Jitter, Shimmer, 추임새 및 침묵 구간 분석
